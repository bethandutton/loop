# Changelog

- Tauri app initialized with React, TypeScript, Tailwind CSS

- macOS Keychain integration for token storage
- First-run onboarding flow (Linear, GitHub, repo setup)
- Settings panel (Cmd+,)
- Command/event layer scaffolded
